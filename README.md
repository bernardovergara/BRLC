# BRLC - Brasil Location Codes

📌 Descrição
'BRLC' é um pequeno utilitário em Python para consultar códigos e nomes de estados (UF) e municípios brasileiros usando uma base de dados em planilha Excel (db.xlsx) disponibilizada pelo ibge. O desenvolvi apenas pelo fato de repetidas vezes necessitar de utilizar essas funções em multiplos projetos. Ele oferece funções para converter entre abreviações, IDs de UF e nomes, além de buscar IDs ou nomes de municípios.

---
